## Design notes

### C7a — optional primitives

`Int?` / `Bool?` emit as `aura_opt_i64` / `aura_opt_bool` (`has` + `value`). Null/wrap/coerce; `== null` via `.has`; `!!` / `?:`. `Map.get` returns `Int?`.

### C7b — field Array GC

Heap classes with Array fields get `aura_gc_alloc_full` + optional `dtor` (free each Array buffer) and `mark_extras` (mark Array-of-class elems).

### C7c — field Array move-out

`return this.items`, bare `return items`, `val b = h.items`, and assign from Array field copy then zero the field so dtor and caller do not share the buffer. Non-destructive field _reads_ that should not move still need a future borrow/`ref` type.

### C7e — Set

Linear `Set` of String parallel to Map (Array of keys + contains/add/remove). No generic Set yet.

### C7h — Array of interface

MVP default: keep reject with clear diagnostic. Alternative: erase to fat pointer (iface tag + data) so Array elem size is stable. Decision recorded in debts + guide.
