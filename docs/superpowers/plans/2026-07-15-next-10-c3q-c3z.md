## Overview

Ship ten small, verifiable P3 slices after **C3p**. Order prefers: hygiene → memory safety without full GC → package/type completeness → string iteration → GC foundation → std prelude. Each task is one commit, sized like recent C3 work (≈1 session, ≤~5 core crates + corpus/docs).

## Architecture decisions

1. **Stay on C backend** — no LLVM in this batch; GC/runtime evolves in `runtime/aura_rt.c` + codegen.
2. **Debt-first, then P3 pillars** — clear Open debts before registry/LLVM; GC is last foundation, not a mega-PR.
3. **No full iterator protocol yet** — expand `for-in` to `String` only; Iterable stays deferred.
4. **Registry still deferred** — only lock-graph hygiene if needed; no semver/registry server.
5. **Commit message convention** — `feat(compiler|cli|runtime): C3x …` + body optional; mirror C3n–C3p.

## Dependency graph

```text
C3q (parens) ─────────────────────────────── independent
C3r (Array.pop) ──► C3t (Array free) ────── uses same Array layout
C3s (ex free) ────────────────────────────── independent runtime
C3u (Alias.Type) ──► nicer multi-pkg demos
C3v (type C symbols) ──► required before multi-pkg classes collide in GC era
C3w (for-in String) ──────────────────────── independent language
C3x (GC runtime MVP) ──► C3y (class as GC ref)
C3y ──► C3z (std prelude can stay parallel but after packages stable)
```

**Sequential spine:** C3q → C3r → C3s → C3t → C3u → C3v → C3w → C3x → C3y → C3z  
**Safe parallel pairs** (if multi-agent): (C3q ‖ C3s), (C3u ‖ C3w) after C3r/C3t if Array free not shared.

### Task 1 — C3q: codegen equality parentheses

**Description:** Stop emitting double parentheses around equality in conditions so clang does not warn `-Wparentheses-equality` (e.g. `if ((t == INT64_C(10)))` → `if (t == INT64_C(10))`).

**Acceptance criteria:**
- [ ] Generated C for `==` / `!=` in `if`/`while` has a single paren level from the condition, not nested `( (a == b) )`
- [ ] Existing corpus still builds with `cc` without new warnings of this class
- [ ] Debt entry “C equality emits extra parentheses” moved to Resolved in `agents/debts.md`

**Verification:**
- [ ] `cargo test -p aura-codegen` (or workspace)
- [ ] `cargo run -p aura-cli -- build corpus/hello/main.aura -o target/aura/hello` and inspect emit for equality ifs
- [ ] Optional: compile with `-Wparentheses-equality -Werror` on one sample

**Dependencies:** None  
**Files likely touched:** `crates/aura-codegen/src/expr.rs` (and/or `stmt.rs`), `agents/debts.md`, `docs/roadmap.md`, `README.md`  
**Scope:** XS–S  
**Commit:**
```text
fix(compiler): C3q drop redundant equality parens in C emit
```
